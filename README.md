# Streamlit_CreditCalculator_EA
Приложение на Streamlit для расчета аннуитетных и дифференцированных платежей по кредиту

# Кредитный калькулятор

Приложение для расчета кредитных платежей с визуализацией графика погашения.

## Функциональность

- Расчет аннуитетных и дифференцированных платежей
- Ввод суммы кредита, процентной ставки и срока
- Детальный график платежей с разбивкой на проценты и основной долг
- Отображение общей суммы выплат и переплаты
- История расчетов

## Дополнительные возможности

- Указание даты первого платежа с автоматическим расчетом дат для каждого платежа
- Обработка ошибок ввода
- Сворачиваемая таблица с графиком платежей
